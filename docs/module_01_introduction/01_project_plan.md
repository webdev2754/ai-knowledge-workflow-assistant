# Modul 1 – Projektplanung (Kapitel 1)

## 1. Use Case Definition
AI Knowledge & Workflow Assistant:
- Kann Dokumente lesen (PDF, Text, Webseiten)
- Kann Fragen beantworten (RAG)
- Kann Tools nutzen (Agenten)
- Kann Workflows planen
- Kann Entscheidungen treffen
- Nutzt Finetuning für Domänenwissen

## 2. Use Case Evaluation
- Nutzer: Entwickler, Teams, Unternehmen
- Nutzen: Automatisierung, Wissensmanagement, Analyse
- Risiken: Halluzinationen, Kosten, Latenz
- Anforderungen: RAG, Agenten, Evaluierung, Monitoring

## 3. Setting Expectations
- MVP: RAG + einfacher Chat
- Version 2: Agenten + Tools
- Version 3: Finetuning + Monitoring

## 4. Milestone Planning
- M1: Modell auswählen
- M2: Prompt-Stack
- M3: RAG
- M4: Agenten
- M5: Finetuning
- M6: Dataset Engineering
- M7: Optimization
- M8: Final Architecture

## 5. Maintenance
- Datenupdates

- Monitoring & Feedback
